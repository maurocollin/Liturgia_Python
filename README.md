O **Liturgia Python** é uma aplicação desktop desenvolvida para facilitar o acesso à Liturgia Diária Católica. Através de técnicas de **Web Scraping**, o app busca as leituras, salmos e o evangelho do dia em tempo real, apresentando-os em uma interface moderna e intuitiva.

---

## 🚀 Funcionalidades

- **Acesso em Tempo Real:** Captura os textos litúrgicos diretamente de fontes oficiais via Web Scraping.
- **Interface Moderna:** Desenvolvido com `CustomTkinter` para oferecer um visual Dark/Light Mode nativo e responsivo.
- **Leitura Formatada:** Utiliza `textwrap` para garantir que os textos longos sejam exibidos de forma organizada e legível.
- **Automação de Data:** Identifica automaticamente a data do sistema para buscar a liturgia correspondente ao dia.

---

## 🛠️ Tecnologias Empregadas

Este projeto destaca minhas competências em:

* **Python 3**: Linguagem base para toda a lógica de processamento.
* **BeautifulSoup4 (bs4)**: Extração e tratamento de dados de páginas HTML.
* **Requests**: Gerenciamento de requisições HTTP para coleta de dados.
* **CustomTkinter**: Criação de interfaces gráficas (GUI) com design moderno.
* **Programação Modular**: Organização de código limpo e escalável.

## 💻 Como rodar a aplicação

Siga os passos abaixo para configurar o ambiente e executar o projeto em sua máquina:

### 1. Clonar o Repositório
```bash
git clone [https://github.com/maurocollin/Liturgia_Python.git](https://github.com/maurocollin/Liturgia_Python.git)
cd Liturgia_Python
```
### 2. Criar um Ambiente Virtual (Opcional, mas recomendado)
No Windows:
```bash
python -m venv venv
.\venv\Scripts\activate
```
No Linux/Mac:
```bash
python -m venv venv
source venv/bin/activate
```

### 3. Instalar as Dependências
```bash
pip install -r requirements.txt
```
Nota: Caso não tenha o arquivo requirements.txt, instale manualmente: > pip install requests beautifulsoup4 customtkinter

### 4. Executar o App
```bash
python app.py
```
